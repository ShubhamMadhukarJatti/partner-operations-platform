# Sharkdom Docs

## Tech Stack

Package manager : `PNPM`

- [Nextjs](https://nextjs.org/)
- [Tailwindcss](https://tailwindcss.com/)
- [Shadcn-ui](https://ui.shadcn.com/)
- [Vercel](https://vercel.com/)
- [API Docs](https://dev.sharkdomapi.com/swagger-ui/index.html#/)

## Getting Started

First, run the development server:

install pnpm

add .env.local file

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Docker

Use the centralized runner for Docker Compose:

```bash
python run.py up
python run.py logs
python run.py down
```
